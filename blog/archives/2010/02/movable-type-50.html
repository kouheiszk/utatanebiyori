Movable Type 5.0 にアップグレード

なぎ (2010年2月 4日 01:08) | コメント(6) | トラックバック(0)
Movable Type 5が出てたからバージョンアップしてみました。
4.2から5.0へのアップデートは何もないサーバーに4.2をインストールするよりもすんなりいきました。

やり方はMovable Type 5 第一回勉強会のサイトを参考にしました。
そこにも書いてあったけど、ウェブサイトのURLが http://uzusionet.com/ に...。

このブログのURLはウェブサイトの utatanebiyori っていうサブドメインの blog ディレクトリ以下ってことになってた。
つまりhttp://utatanebiyori.uzusionet.com/blog/ってこと。
最終的に正しいURLになってたけど、気持ち悪いから直しました。

画像のパスとかどうなってるんだろう...気になります。

ついでにテーマも変更。
http://vicuna.jp/ のデザインを使わせてもらいました。
いろいろと改造していく予定。

なんかそんなことを1年前にも言ってた気がします。
三日坊主にならないように気をつけないと。
最近rhacoってプラグイン兼フレームワークを使って大学のコミュニティーサイトを作ろうって思って...
uzusionetのphpのmbstringの設定がSJISでいろいろと文字化けして困ってます＾＾；

そもそもマルチバイト文字のXSSに貧弱性があるのになんでSJIS使うんだ?
って思ったけど、きっと僕よりもこのサーバーを設定している彼らの方が技術力があるわけだから、何か理由があるんだろうなっておもって、いろいろrhacoのソースをいじくってます。

...__ini__.phpじゃぁ設定できなかった。

そもそもMovable Typeで作った方がいいのかって、このMT5入れた瞬間に思った。

いや、PHP使えるようになりたいし!!がんばります。

とりあえず、データベースの文字コードもいろいろ真っ白な今だから、開発よりも調査が先ですね。
