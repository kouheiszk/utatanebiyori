**TeXインストーラ3を使った導入方法 [#b82594f4]
ネットでTeXのインストール方法を調べてみると、ダウンロードからして結構面倒みたいです。そこでここでは「TeXインストーラ3」というソフトを使ったファイルのダウンロードからインストール、W32TeXの使用方法までを紹介します。

--ダウンロードとインストール
まず下記URLより、TeXインストーラ3のパッケージをダウンロードしてください。
[[http://www.ms.u-tokyo.ac.jp/~abenori/mycreate/index.html:http://www.ms.u-tokyo.ac.jp/~abenori/mycreate/index.html]]
zipで圧縮されています。解凍すると、「kakuto3」というフォルダが生成されるので、その中の「kakuto3.exe」という実行ファイルを起動してください。
基本、あとは「次へ」のクリックだけで進みますが、「失敗」と出た場合は必要なパッケージをダウンロードしてくる場所がリンク切れになっている可能性があります。そういうときは、「戻る」ボタンをクリックし、失敗した各種ソフト(W32TeX,dviout,GhostScript,GSView）のダウンロード先のURLを変更してください。何度か試していれば全部成功する筈です。
詳しくはreadme.txtを参照してください。TeXインストーラ3のパッケージが置いてある横にもあります。

--W32TeXの使い方
pTeXというソフトをMicrosoft Windows 32ビットプロセッサ (x86, IA-32) 環境 (Windows 95, 98, Me, NT, 2000, XP, Vista,...) 用に移植したものがW32TeXです。コマンドプロントを使って操作します。
最初にtexファイルやdviファイルを置くフォルダをどこかに作ってください。My Documentsあたりがいいかと思います。ここでは仮にCドライブのルート直下にMy Documentsを設定していて、その中に「tex」というフォルダを作った場合について話します。
まずコマンドプロントで、
 C:
 cd \My Documents\tex
と打ち込み、カレントディレクトリをtexフォルダにしてください。次に、
 notepad ***.tex
 （***はtestでもなんでもかまわないです。ただし、コマンドプロントで打ち込むので英字の方が後で楽です）
と打ち込み、メモ帳を起動してください。「新しく作成しますか？」と聞いてくるので「はい」をクリックしてください。あとは、
 \documentclass{jarticle}
 \begin{document}
 
 ・・・文章もしくは数式・・・
 
 \end{document}
みたいな感じにメモ帳でコマンドを書き、上書き保存してください。
次にコンパイルです。コマンドプロントで、
 platex ***.tex
と打ち込めばOKです。生成されたdviファイルはtexファイルと同じフォルダ（ここではtexフォルダ）に保存されます。
 ***.dvi
と打ち込むと、dvioutというソフトが立ち上がり、文書のプレビューを見ることができます。また、
 dvipdfmx ***.dvi
と打ち込めば、pdfファイルに変換することもできます。pdfファイルも同様に、texフォルダに保存されます。

あとはGMailなどでネットに上げたり、USBメモリーに入れたりして、IEDやCCで、psファイルに変換、もしくはpdfファイルにして持ち込みそのまま印刷、などして先生に提出してください。USBメモリーの使い方はIEDのWebページに載ってます。
[[http://www.ied.ice.uec.ac.jp/:http://www.ied.ice.uec.ac.jp/]]

なお、今回はTeX Wikiというページを参考にしました。詳しい説明はこちらを参照してください。
[[http://oku.edu.mie-u.ac.jp/~okumura/texwiki/:http://oku.edu.mie-u.ac.jp/~okumura/texwiki/]]
TeXのコマンドは「TeX 入門」てとこを見ながらやれば慣れてくと思います。多分。

**VMwareを使った導入 [#gc47cdb3]
VMware Ubuntuをインストールしていることが前提です。
以下のコマンドを入力することで自動的にソフトにインストールが始まり、LaTeXを導入することができます。
 % sudo apt-get install ptex-bin
 % sudo apt-get install dvipdfmx
コンパイルなどのコマンドも授業と同じでそれぞれ以下のようになります。
 % platex sample.tex
 % xdvi sample.dvi &
 % xdvipdfmx sample.dvi
上からtexファイルをコンパイルするコマンド、出来上がったdviファイルを確認するコマンド、dviファイルをpdfに変換するコマンド。
