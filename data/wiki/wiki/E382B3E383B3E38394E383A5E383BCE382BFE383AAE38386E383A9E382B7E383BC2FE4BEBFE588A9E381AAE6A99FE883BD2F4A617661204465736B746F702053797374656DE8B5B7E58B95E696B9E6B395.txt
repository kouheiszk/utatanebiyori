* Java Desktop System起動方法 [#pcefc456]

Java Desktop System（JDS((Java Desktop System の略))）は、サン・マイクロシステムズのSolaris上のデスクトップ環境である。

平均的なコンピュータユーザーが使い易い（つまり、Microsoft Windows によく似た）システムを提供しており、オフィススイート、ウェブブラウザ、電子メール、カレンダー、インスタントメッセンジャーなどのソフトウェアが全て揃っている。

** カトウ式JDS起動法 [#nc02a7a3]

- オギノ式を尊敬してるのでカトウ式と名づけます。

まず、firefoxを起動。何でもいいので画像とかを"名前を付けて保存"。
ダウンロードウインドウが開いてダウンロード状況が出る。
ダウンロードが終わったら、ファイルを右クリックして"保存フォルダを開く"をクリックするとＪＤＳが起動する。
終了方法とかはいまいち分かりません。管理者から見ればある種のセキュリティーホールに対する攻撃とも見なせるのでご利用は計画的に。

** xtermのコマンドからの起動方法 [#q9a40032]

- まず、xtermがフォアグランドの状態で、画面の適当なところを左クリックすると、なんかいっぱい出るので、一番下のQuit twmを選択する。すると、x window の赤い枠が消える。先輩曰く、枠があるといろいろめんどくさいらしいので、この作業で消す。
- 次に、xtermで次のチートコードを入力する。

 % gnome-session

※一度実行するとカトウ式でのJDS起動ができなくなります。((ただしQuit twmを実行せずにgnome-sessionを実行すると起動できる。))
※x window の枠を消した状態でxtermを見失うと、もれなくtelnetの使い方をサーバ管理者から教えてもらえるかもしれません。

** ログイン時に自動起動させる [#xf432dfb]

ホームディレクトリに以下内容で二つのファイルを作成し、実行権限を与える。

.xsession
#code(sh){{
#!/bin/csh
# no -f in the previous line so .cshrc gets run to set $PATH

exec /usr/bin/gnome-session
afterstep
}}

.xinitrc
#code(sh){{
#!/bin/sh

exec $HOME/.xsession
}}
